- HEM: Hemangiomas (peripheral nodular enhancement, fill-in).
- APHE_WO: Arterial Phase HyperEnhancement (APHE) WITH Washout. (Classic HCC pattern, LR-5).

- RIM_ATYP: Rim enhancement, Peripheral washout, Targetoid appearance, Cholangiocarcinoma features, Metastases, Treated lesions/Scars, or any other atypical finding not fitting the above.

3. QUANTITY MAPPING:
- Specific number (1, 2, 3...) -> Write the number.
- Words like "multiple", "innumerable", "many", "foci" (plural without number) -> Multiple
