Capture EVERY distinct visual observation mentioned in the liver parenchyma. You must classify EVERY finding into one of the 5 allowed types based on its imaging features.

MAPPING RULES (Strictly follow these):

1. LOCATION MAPPING (Based on Couinaud Segments):
- Seg 2, 3 -> L_LAT
- Seg 4 -> L_MED
- Seg 5, 8, Dome -> R_ANT

- Seg 1 -> CAUDATE
- Boundary (5/6, 6/7, 5/8) -> R_JUNCTION

- Right Lobe (General) -> R_DIFFUSE
- Not specified -> DIFFUSE

2. TYPE MAPPING (Strictly limited to these 5 categories):
- CYST: Simple cysts, polycystic liver, fluid collections, seromas.
- HEM: Hemangiomas (peripheral nodular enhancement, fill-in).
- APHE_WO: Arterial Phase HyperEnhancement (APHE) WITH Washout. (Classic HCC pattern, LR-5).
- APHE_NoWO: Arterial Phase HyperEnhancement WITHOUT Washout. (Includes LR-3, LR-4, Flash-filling hemangiomas, THAD, Shunts, Perfusion anomalies, and indeterminate arterial foci).
- RIM_ATYP: Rim enhancement, Peripheral washout, Targetoid appearance, Cholangiocarcinoma features, Metastases, Treated lesions/Scars, or any other atypical finding not fitting the above.

3. QUANTITY MAPPING:
- Specific number (1, 2, 3...) -> Write the number.
- Words like "multiple", "innumerable", "many", "foci" (plural without number) -> Multiple

TASK:
1. Break down the report content into individual sentences.
2. Analyze each sentence to check if it describes a focal liver observation.
3. If a finding is identified, determine its Type, Location, and Quantity based on the rules above.
4. Ignore findings clearly stated as belonging to other organs (e.g., kidney cysts).
